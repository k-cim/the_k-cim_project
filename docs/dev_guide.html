API Python & architecture modulaire

  • Importer GridKeep comme bibliothèque : 
    import gridkeep.core.core_engine as gk
  • Lire un fichier JSON : 
    data = gk.load_json("monfichier.json")
  • Accéder à une clé profonde : 
    value = gk.get_key(data, "config.user.settings.theme")
  • Ajouter un parser personnalisé : 
    # Implémenter votre classe MyParser, puis
gk.register_parser("myformat", MyParser)
  • Écrire un plugin de validation : 
    def validate_custom(data):
    # code ici
    return True
gk.register_validator("myplugin", validate_custom)
