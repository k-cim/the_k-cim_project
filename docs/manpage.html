Manpage CLI GridKeep

Commandes principales

USAGE: gridkeep_cli.py [OPTIONS] [FICHIER]

Options principales :
  --read-key            Afficher la valeur d’une clé (chemin complet)
  --write-key   Écrire/mettre à jour une clé
  --search-key          Recherche floue de clé, retourne tous les chemins possibles
  --read-value       Recherche toutes les clés contenant une valeur donnée
  --batch           Appliquer un batch d’actions via un fichier JSON/YAML
  --diff-json          Comparer deux fichiers JSON/TOML/INI/XML, voir les différences
  --patch-json    Appliquer un patch JSON sur une base
  --format             Forcer le format de parsing (json, ini, toml, xml)
  --log                      Activer la journalisation détaillée
  --rollback                 Restaurer l’état précédent du fichier (snapshots log)
  --help                     Afficher l’aide

Exemples d’utilisation

# Lire une clé imbriquée
$ gridkeep_cli.py --read-key "user.settings.display.theme" config.json

# Mettre à jour une valeur dans un fichier TOML
$ gridkeep_cli.py --write-key "pipeline.stage[0].name" "CleanStep" config.toml

# Appliquer un batch d’opérations
$ gridkeep_cli.py --batch my_updates.json config.json

# Diff JSON/TOML
$ gridkeep_cli.py --diff-json prod.json staging.json
