FAQ

Questions fréquentes

  • Quels formats sont supportés ?
    JSON, INI, TOML, XML (bientôt YAML, JSObject…)
  • Est-ce que GridKeep fonctionne sur Windows/macOS/Linux ?
    Oui, tout est multiplateforme.
  • Puis-je utiliser GridKeep pour éditer des fichiers massifs ?
    Oui, il est optimisé pour la performance sur gros fichiers.
  • Comment ajouter un plugin ?
    Voir la doc développeur.
  • Les logs sont-ils consultables ?
    Oui, tous les logs de modifications sont sauvegardés, consultables, et rollbackables.
  • Peut-on contribuer ?
    Oui, code et site sont open source (voir Contact).
