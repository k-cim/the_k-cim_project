Fonctionnalités

Gestion de fichiers complexe, sans compromis

  • Manipulation directe : JSON, INI, TOML, XML (multi-format, multi-entrée)
  • Lecture/écriture de clés imbriquées, batch, pipes, stdin/stdout natif
  • Rollback, logs, diff, patch, export, conversion, recherche profonde (deep search)
  • Sécurité : jamais de suppression automatique, contrôle total des modifications
  • Automatisation : scripts CLI, batch processing, hooks, intégration CI/CD
  • UI Graphique : édition visuelle, sélection avancée, drag & drop
  • Extensible : architecture plugins, modules parsers, visualisation custom
  • Synchronisation live : WebSocket, mode cloud, live diff

Points forts

  • Performance extrême même sur fichiers massifs (plusieurs Go)
  • API Python et CLI POSIX, documentation intégrale
  • Multi-plateforme : Linux, macOS, Windows
  • 100% libre, open source, sous GPL v3
  • Site, docs et interface traduits (FR/EN)
